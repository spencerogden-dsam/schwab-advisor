# schwab-advisor

Python client for Schwab Advisor Services API.

## Installation

```bash
pip install schwab-advisor
```

## Usage

```python
from schwab_advisor.client import SchwabAdvisorClient

# Basic usage (API details to be implemented)
with SchwabAdvisorClient() as client:
    pass
```

## Development

This project uses [Poetry](https://python-poetry.org/) for dependency management.

```bash
# Install dependencies
poetry install

# Run tests
poetry run pytest
```
